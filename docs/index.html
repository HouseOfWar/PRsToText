üéØWhat is PRsToText?

PRsToText is a simple, free Python tool that fetches pull request information from GitHub repositories and converts it into easy-to-read text files. It's specifically designed to help stateful AI agents and automated systems track project progress by providing a digestible format of PR history.

‚ú®Features

Simple & Fast

One command to fetch and format all your pull requests into readable text.

Public & Private

Works with both public and private repositories with optional token authentication.

Comprehensive Data

Extracts titles, states, authors, URLs, and timestamps for complete PR context.

Free & Open

Completely free to use with no hidden costs or limitations.

üöÄQuick Start

Installation

git clone https://github.com/lmichaelwar/PRsToText.git
cd PRsToText
pip install -r requirements.txt

Usage

Edit prs_to_text.py and update these lines:

repo_owner = "your-repo-owner"  # e.g., "octocat"
repo_name = "your-repo-name"    # e.g., "Spoon-Knife"

Then run:

python prs_to_text.py

Your formatted pull requests will be saved to {owner}_{repo}_prs.txt

üîëGitHub Token (Optional)

For private repositories or to avoid rate limits, set up a GitHub Personal Access Token:

  1. Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens
  2. Generate a new token with repo scope
  3. Set it as an environment variable:
export GITHUB_TOKEN="your_token_here"
